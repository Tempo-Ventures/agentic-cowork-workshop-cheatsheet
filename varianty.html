Terminál a Desktopová aplikace sdílí jádro. Stejné nastavení, stejná historie konverzací. Rozhraní jsou různá, agent je stejný.
Cowork není Claude Code. Anthropic ho prodává jako samostatný produkt pro netechnické úlohy. Do mapy patří, protože řeší podobné věci pro lidi, kteří nepíšou kód.
Cloud agents nejsou Claude Code. Jsou to Managed Agents na Claude Developer Platform — vlastního agenta si složíte z modelu, nástrojů, MCP a skills a Anthropic ho hostuje v cloud sandboxu. Jiná cesta než interaktivní kódování, ale ke stejnému agentovi.
Routines a Web jsou v preview. Routines i Claude Code na webu (claude.ai/code) jsou research preview, Cloud agents běží v betě — triggery, limity i chování se můžou měnit. Pro produkční nasazení s rezervou.
Web a Desktop se neperou. Web použijete na cizím stroji nebo mobilu, desktop na svém. Dají se kombinovat — historie konverzací se synchronizuje.
